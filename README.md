# slackbot30Minutes

## Installation
``` bash
$ npm install
```

## In .env file
```bash
SLACK=...
PAGE_TOKEN=...
CHANNEL_ACCESS_TOKEN=...
APIAI=...
```

## Running the application for SLACK
``` bash
$ npm run slack
```

## Running the application for KAKAO
``` bash
$ npm run kakao
```

## Running the application for FACEBOOK
``` bash
$ npm run facebook
```

## Running the application for LINE
``` bash
$ npm run line
```
